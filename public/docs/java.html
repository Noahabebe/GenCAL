Java Documentation

This guide will help you integrate CalScript with Java.

Setup

Save CalScriptParser.java in your project directory and compile with javac CalScriptParser.java.

Usage

public class Main {
    public static void main(String[] args) {
        String script = """
        Event: "Meeting" -> Day: "Monday" @ Time: "09:00"
        Repeat: "Daily Standup" -> Every: "1 Days" @ Time: "10:00"
        """;

        CalScriptParser parser = new CalScriptParser();
        parser.parse(script);
        for (Event event : parser.getEvents()) {
            System.out.println(event);
        }
    }
}
