C++ Documentation

This guide will help you integrate CalScript with C++.

Setup

Save CalScriptParser.cpp in your project directory and compile with g++ CalScriptParser.cpp -o CalScriptParser.

Usage

#include "CalScriptParser.cpp"

int main() {
    std::string script = R"(
    Event: "Meeting" -> Day: "Monday" @ Time: "09:00"
    Repeat: "Daily Standup" -> Every: "1 Days" @ Time: "10:00"
    )";

    CalScriptParser parser;
    parser.parse(script);
    auto events = parser.getEvents();
    for (auto& event : events) {
        event.print();
    }

    return 0;
}
