Python Documentation

This guide will help you integrate CalScript with Python.

Setup

Save CalScriptParser.py in your project directory.

Usage

from CalScriptParser import CalScriptParser

script = """
Event: "Doctor's Appointment" -> Day: "Tuesday" @ Time: "15:00"
Repeat: "Daily Exercise" -> Every: "1 Days" @ Time: "07:00"
"""
parser = CalScriptParser()
parser.parse(script)
events = parser.get_events()
print(events)
